# OAuth Abuse IOCs

**Goal:** Indicators of compromise and hunting guidance for OAuth consent phishing, malicious Entra ID/Azure AD app registrations, and related token-theft and phishing campaigns (e.g. Power Apps portals, GitHub/GitLab landing pages).

**Contents:** `Microsoft-Intel-OAuth.md` — malicious OAuth client IDs, initial redirection URLs (landing pages), malicious files observed, behavioral indicators (PowerShell, DLL sideloading), phishing themes, and hunting signals (OAuth URL patterns, redirect chains, LNK→PowerShell→DLL).
