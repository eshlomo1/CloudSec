# SkillScan

**Goal:** Scan AI skill files and URLs for malicious patterns: prompt injection, suspicious URLs, code execution, and obfuscation. Supports CLI, local server, and web UI workflows.

**Contents:** `skillscan.py` (Python CLI), `skillscan.ps1` / `skillscan.bat` (wrappers), `skillscan_server.py` / `skillscan_server.ps1` (server), `skillscan_web.html` (web UI).

---

## Usage

Run from this folder; see repo root and [../README.md](../README.md) for requirements. Python 3.x recommended for the CLI and server.
