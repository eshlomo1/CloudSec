# Security Policy

## Supported Versions

Users are encouraged to use the latest version of the toolkit.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this toolkit seriously. If you find a security vulnerability within the tools or scripts provided here, please report it via one of the following methods:

1. **Email:** [Elli](mailto:eshlomo9@gmail.com)
2. **GitHub Issues:** Use the private vulnerability reporting feature on GitHub.

Please provide a detailed summary of the vulnerability, including steps to reproduce. We will acknowledge your report within 48 hours and provide a timeline for fixes.
